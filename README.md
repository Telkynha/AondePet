Bem-vindo ao AondePet

Esse é um projeto feito baseado no tema MAAS, com o objetivo de ajudar todos os pets e os seus donos.
Nesse app é possível visualizar os animaiszinhos facilmente, todas as informações essênciais estarão aqui para você buscar/ajudar um novo amigo!
Te convidamos a explorar os Pets que estão registrados em nosso sistema, você pode se apaixonar!

Feito em Kotlin, JetpackCompose, Firebase e Coil Compose, na IDE Android Studio Koala
